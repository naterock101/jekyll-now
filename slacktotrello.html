To use the badass slack to trello integration built by Nate...

Format:

/trello {card_name} | {person_to_assign} | {label_to_assign} | {description}

Notes:

*besides the {card_name} everything is optional however you have to still put arguments in the correct place seperated by pipes ||

e.g. /trello cool new card ||| this is a cool new card

**[person_to_assign] can only be =

nate, nathan, brac, zern, zernyu, harris, garrett, caroline, dan, gabe, gabriel

***[label_to_assign] can be the label OR the color =

backend lime
frontend pink
bug red
firmware sky/light blue
system black
design yellow
dev ops/devops green
mechanical orange
improvement blue
